What is MCPW

MCPW is a tool for building and editing Minecraft resource/texture packs.
Originally forked from Resource Pack Workbench.

It shows you the pack structure and lets you replace or edit the files.

What it can do

  • Create, modify and remix packs
  • Edit assets using your favorite editors (GIMP, Audacity...)
  • Edit mod assets (not only Vanilla)
  • Add custom sounds for "/playsound"
  • Add custom languages
  • Import and export stiched textures for easy editing
  • Export the pack directly to your Minecraft folder, for easy testing, as zip or folder

Creating a project

MCPW can create a blank project, or a project from existing resource pack.

Configuring editors

On Windows, you'll have to locate your "exe". On Linux, just the binary name would suffice.

Runtime log

The log view is available from the Help menu.

It can help with analyzing bugs, and you can check there what exactly is MCPW doing.

MCPW also saves it's log into a file in the working directory (~/.mcMCPW)

Stitcher / Unstitcher

MCPW contains a utility for stitching and un-stitching textures.

Do not confuse it with the old Mojang-provided tool - this is something different.

It's useful when you feel like editing more textures at once, and don't want to reopen the editor all the time. But not everyone will like it, of course.
