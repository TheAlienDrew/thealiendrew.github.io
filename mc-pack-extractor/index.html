Minecraft Pack Extractor

What is MC Pack Extractor

Home Downloads Support Contribute Videos Screenshots

MC Pack Extractor is a tool for extracting the default Minecraft resource/texture packs.


Double click the script, pick a version, wait a little, and BAM you have yourself a zip file.

What it can do

  • Extract the default textures from Minecraft (releases only, at the moment)

Recomended programs to have

  • 7-Zip, as it is a very useful file zip utility (and my scripts work best with it)

Coming Soon

  • Displaying all versions a user has downloaded before allowing them to choose
  • Going to include support for extracting sounds with the textures (and also as separate options), but for now use this
